# CS Fundamentals Academy

Welcome to **CS Fundamentals Academy** — Learn CS to Become a Better Vibe Coder.

Master computer science fundamentals — how computers work, programming, web dev, data structures, algorithms, and more — through interactive examples built for AI-assisted developers.

## Overview

Become a better vibe coder. Directing an AI coding agent well means reading its output critically — spotting a bad data structure choice, recognizing a security gap, knowing why a request failed. None of that comes from prompting harder. It comes from understanding what's actually happening underneath.

## Features

- **12 Modules, 70+ Topics**: From how a CPU executes an instruction to designing software architecture.
- **100% Interactive**: Write code. See it run. Instantly. A live HTML/CSS/JS editor with instant preview, plus a terminal simulator for practicing real commands risk-free.
- **Progress Tracking**: XP, streaks, and badges that actually track learning.
- **Responsive Design**: Optimized for all screen sizes, from mobile to desktop.

## Running Locally

To run the project locally, simply serve the directory using a local web server. For example:

```bash
npx serve .
```

Then, open `http://localhost:3000` in your web browser.
